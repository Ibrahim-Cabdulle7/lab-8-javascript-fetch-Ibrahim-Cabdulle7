This lab uses [JSONPlaceholder](https://jsonplaceholder.typicode.com/) - a free fake REST API for testing and prototyping.

**Endpoint:** `https://jsonplaceholder.typicode.com/posts`

## How to Test

1. Open `step-01.html` in a web browser
2. Click "Fetch Data" to retrieve posts from the API
3. Observe the loading indicator while data is being fetched
4. View the retrieved posts displayed on the page
5. Click "Clear Data" to remove the displayed content
6. Test error handling by modifying the API URL to an invalid endpoint
